# SpicyVim
My nvim config

Try it with docker:
```
 docker run -w /root -it --rm alpine:edge sh -uelic '                                                                                                                                                                                                                                                                   ─╯
  apk add git neovim ripgrep alpine-sdk fzf --update
  git clone https://github.com/albertocerrone/SpicyVim ~/.config/nvim
  cd ~/.config/nvim
  nvim
'
```
